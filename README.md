# SMART DANCE · Летние городские сборы 2026

Лендинг летней танцевальной лаборатории SMART DANCE, две смены:

- **Смена I · июль** — 06.07 — 31.07.2026 · пр. Независимости, 179, Минск
- **Смена II · август** — 03.08 — 28.08.2026 · ул. Саперов, 5 · ФОК «Атлант»

Программа в обеих сменах одинаковая, отличается только адрес. Доступна запись на 1, 2, 3 или 4 недели внутри одной смены.

## Стек

- React 18 через UMD CDN
- Babel Standalone для JSX в браузере
- Ванильный CSS с кастом-свойствами

Никакого бандлера — всё работает как статические файлы.

## Локальный запуск

```sh
python3 -m http.server 8080
open "http://localhost:8080/SMART%20DANCE%20Landing.html"
```

## Структура

| Файл | Что делает |
|------|------------|
| `SMART DANCE Landing.html` | Точка входа |
| `landing.jsx` | Все секции лендинга |
| `styles.css` | Оформление |
| `tweaks-panel.jsx` | Панель настройки темы/палитры |
